Easy D

A multi feature GUI application

Description

Easy-D is modern multi-feature desktop application developed with the aim to integrate multiple applications into one, whether it's your favorite media, or you want to go maps, Easy-D is there at your service. Easy-D is python based, with the rich set of python modules induced in the application development process, Easy-D utilizes multiple python modules. Being a Tkinter GUI based application, it is easy to use, develop, and is cross platform with portable compact code design.

Easy-D is combination of the following applications into one desktop app:-

  1. Music Player
  2. Maps-Go
  3. AuTo ChAt
  4. Video Player

All the above functionalities works in sync to provide an extensible area to explore around with. Music on the go, lost??? Don't worry Easy-D got you covered with the Maps-Go, a real-time geolocation maps (Google-Server) with the coordinate functionality to pin, search and explore the parts around the world.
